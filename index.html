Outlook2myleave

Takes Outlook calender events and creates them in myLeave

Download this project as a .zip file Download this project as a tar.gz file

This project aims to read Outlook calender events and automatically input the correct information into myLeave, a digital time sheet service. It will enter Walkin shifts, Incidents, Tasks, and blank time into myLeave, so keep your calender updated! 

Updates will be pushed to GitHub occasionally, ask Zach or I if you want more info or update information
